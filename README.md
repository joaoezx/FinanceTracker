# Finance Tracker

Aplicação web para organização financeira pessoal.
O sistema permite importar gastos de cartões de crédito, controlar investimentos e simular financiamentos com amortização.

## Objetivo

Este projeto foi desenvolvido pensando auxiliar na organização financeira, e demonstrar uma aplicação com arquitetura em três camadas:

- Front-end
- Back-end
- Banco de dados

O objetivo principal é auxiliar no controle financeiro,
permitindo o acompanhamento de gastos de cartão de crédito,
investimentos e dívidas.

## Tecnologias

### Front-end
- React
- TypeScript
- Chart.js

### Back-end
- Node.js
- NestJS
- REST API

### Banco de dados
- PostgreSQL
